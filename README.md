# Cape-Wildlife-Rescue

Basic HTML website for cape wildlife Rescue, a non-profit dedicated to recuing and rehabilitating injured wildlife in CapeTown.

##Pages
-'home.html', 'about.html'. 'projects.html'. 'get-involved', 'donate.html'

##Setup
1.keep all 'html' files+images in one folder
2.open 'home.html' in a browser

##Stack
HTML5, CSS3 | Colours: #3E8E41, #0077BE | Fonts: lato, Merriweather
